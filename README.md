# nyc-restaurant-inspection-analysis
## 📊 Overview
This project analyzes restaurant inspection data in New York City to identify patterns in food safety, violations, and restaurant performance across boroughs and cuisine types.

This project was completed using a real-world styled dataset and follows a typical data analyst workflow from data cleaning to visualization.

The goal is to uncover insights that can support better public health decisions and targeted inspections.

---

## 🎯 Objectives
- Identify common health violations  
- Compare restaurant performance across boroughs and cuisines  
- Analyze grading patterns (A, B, C)  
- Explore trends in inspections over time  

---

## 🛠 Tools Used
- Excel (data cleaning and analysis)  
- SQL (data querying)  
- Power BI (data visualization)  
- PowerPoint (presentation of insights)  

---

## 💡 Key Insight Areas
- Most frequent violations  
- High-risk boroughs and neighborhoods  
- Poor-performing cuisine types  
- Trends in food safety over time
